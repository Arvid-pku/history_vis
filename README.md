# World History Timeline

An interactive visualization of world history from 3000 BC to 2000 AD, spanning 8 major regions across 5000 years of human civilization.

## Features

- **8 Regions**: East Asia, Southeast Asia, Europe, Middle East, South Asia, Central Asia, Africa, Americas
- **5000 Years**: From the Bronze Age to the Modern Era
- **Bilingual**: English and Chinese language support with explicit active-language state, tolerant restored language links, and stable current-view position while switching languages
- **Interactive**: Hover or focus a dynasty/civilization to preview it and highlight contemporaries; focus numbered events for event previews; click or press `Enter` / `Space` for details, with inline Wikipedia links in descriptions and known-entity detail lists where available
- **Connection Lines**: Visualize trade routes, conflicts, and cultural exchanges; arrows and connection-label text can be toggled separately
- **Historical Events**: 184 major world events marked on a dedicated event rail that follows the active region filter
- **Movable Time Ruler**: Optional horizontal ruler aligns all visible regions at a single year; drag it or adjust it from the keyboard, and share links preserve its state
- **Territorial Semantics**: Core territories, frontier spheres, client states, regional provinces, maritime spheres, and contested occupations can be drawn as separate/narrowed fragments instead of misleading full rectangles, including East Asian Western Jin/Wu and Sui/Chen before-and-after reunification phases, Song/Ten Kingdoms, Silla/Goryeo, Later Jin/Ming and Qing China-proper/Manchuria/Taiwan/Mongolia interlocks, ROC/warlord/wartime transition fragments, Manchuria/Manchukuo/Soviet-Northeast transition lanes, PRC China-proper / Northeast / Tibet / Xinjiang western-frontier fragments, Korea's 1945-48 liberation/occupation split, Japan's Taisho / imperial Showa / Allied occupation / postwar constitutional phases, and early East Asian China Neolithic / Southern Cultures as adjacent whole lanes, steppe-Gojoseon, whole Jomon-Yayoi archipelago lanes with peninsula contact links, and Van Lang / Au Lac / Nanyue / Chinese-rule Vietnam cultural-owner interlocks; early European Aegean/Minoan/Mycenaean/polis, Etruscan, Celtic Gaul, Iberian/Celtiberian, Celtic Britain, Germanic/Lusatian/Slavic/Scythian cultural-owner lanes, Roman Republic/Empire/Late Roman Italy-Greece-Gaul-Hispania-Britain provincial lanes plus true frontier/successor fragments, and post-Roman Italy's Byzantine reconquest / Byzantine Exarchate / Lombard / Frankish sequence; WWII France/Poland/Italy occupation and resistance lanes; Cold War USSR core vs Warsaw Pact Poland influence; Sumer/Gutian/Ur III/Isin-Larsa/Kassite/Post-Kassite Babylonian continuity, Akkadian/Elamite, Old Babylonian/Elamite, Middle/Neo-Elamite, Neo-Hittite, Assyrian/Neo-Babylonian, and staged Achaemenid conquest fragments; ## Controls

| Feature | Description |
|---------|-------------|
| **Zoom** | Use slider, presets, +/- buttons, keyboard shortcuts, or mobile pinch gestures; preset state follows the current scale |
| **Jump to Year** | Enter any year (e.g., "500", "1066 AD", "500 BC", "公元前500") |
| **Era Buttons** | Quick jump to Bronze, Classical, Medieval, or Modern era |
| **Time Ruler** | Toggle a horizontal ruler from the display layers, drag it to any year, or focus it and use arrow keys / Page Up / Page Down to move it |
| **Search** | Find dynasties by name, trim restored query links, jump to the first visible match, and clear back to the search field |
| **Region Filter** | Show/hide specific regions while preserving the current view position; restored links tolerate region keys, full labels, short labels, and Chinese names |
| **Category Filter** | Filter by empire, kingdom, republic, caliphate, dynasty, culture, nomadic, colonial, or league/confederation entities with localized accessible names; restored links tolerate category keys, current labels, and legacy aliases such as `Confederation` / `联邦` |
| **Dark Mode** | Toggle dark theme with pressed-state feedback and shareable state |
| **Full Page** | Distraction-free viewing mode that preserves the current center year and horizontal position; active previews clear, focus moves to the visible exit control, and focus returns on exit |
| **Compare** | Compare two regions side by side with the same robust region-link restoration |
| **Export** | Save as PNG when the image renderer is available; fall back to the print dialog |
| **Share** | Copy shareable URL with current view state, including center year, horizontal timeline position that stays current after manual scrolling, active compare pairs, dark mode, ruler state, explicit empty region selections, clamped zoom values, malformed URL recovery, and failure feedback |
| **Help** | Open a bilingual guide with stable dialog labeling that reflects the current controls, keyboard behavior, and dataset size |

## Keyboard Shortcuts

- `Ctrl/Cmd + +` : Zoom in
- `Ctrl/Cmd + -` : Zoom out
- `Ctrl/Cmd + 0` : Reset zoom
- `Arrow keys` : Scroll vertically or horizontally
- `/` : Focus search
- `Enter` / `Space` : Open the focused polity or world-event detail panel
- `Tab` / `Shift+Tab` : Move through links and actions inside open detail, event, Help, and Compare dialogs
- Time ruler: drag the horizontal line, or focus it and use `↑` / `↓`, `Shift+↑` / `Shift+↓`, `Page Up` / `Page Down`, `Home`, and `End`
- Hover or focus a polity block to preview it and highlight contemporaries; focus numbered event markers from the active regions to preview world events
- Compare panel: opens as a keyboard dialog, traps focus while open, and returns focus to the Compare button when closed
- Symbol and compact controls such as region filters, display-layer toggles, era jumps, command buttons, zoom, help, legend, mobile controls, and full-page exit have descriptive keyboard/assistive labels plus matching expansion/hidden/focus state where applicable
- `Escape` : Close panels and return focus / Exit full page mode
